Declaración de Accesibilidad

Nuestro compromiso con la inclusión digital

Nuestro Compromiso

We Build Pillars está comprometido con garantizar la accesibilidad digital para personas con discapacidades. Mejoramos continuamente la experiencia del usuario para todos y aplicamos los estándares de accesibilidad relevantes.

Como organización dedicada a la equidad digital para comunidades subrepresentadas, creemos que la accesibilidad no es opcional, es esencial. Todos merecen igual acceso a la información y servicios en línea.

Estado de Conformidad

Nos esforzamos por cumplir con las Pautas de Accesibilidad para el Contenido Web (WCAG) 2.1 en el Nivel AA. Estas pautas explican cómo hacer que el contenido web sea más accesible para personas con discapacidades y más fácil de usar para todos.

Características de Accesibilidad

Hemos implementado las siguientes características de accesibilidad en nuestro sitio web:

  • Navegación por Teclado: Toda la funcionalidad es accesible mediante teclado
  • Enlaces de Salto: Enlaces para saltar al contenido principal para usuarios de lectores de pantalla
  • HTML Semántico: Jerarquía de encabezados y regiones de referencia adecuadas
  • Texto Alternativo: Texto alternativo descriptivo para todas las imágenes significativas
  • Contraste de Color: El texto cumple con los requisitos de contraste AA de WCAG (mínimo 4.5:1)
  • Diseño Responsivo: El contenido se adapta a diferentes tamaños de pantalla y niveles de zoom
  • Indicadores de Enfoque: Estados de enfoque visibles para elementos interactivos
  • Etiquetas de Formulario: Todos los campos de formulario tienen etiquetas asociadas
  • Etiquetas ARIA: Soporte de tecnología asistiva donde es necesario
  • Fuentes Legibles: Texto del cuerpo de mínimo 16px con tipografías legibles

Compatibilidad con Tecnología Asistiva

Nuestro sitio web está diseñado para ser compatible con:

  • Lectores de pantalla (NVDA, JAWS, VoiceOver, TalkBack)
  • Software de magnificación de pantalla
  • Software de reconocimiento de voz
  • Navegación solo con teclado

Limitaciones Conocidas

Aunque nos esforzamos por lograr una accesibilidad completa, algunos contenidos pueden tener limitaciones:

  • Algunos documentos PDF antiguos pueden no ser completamente accesibles
  • El contenido de terceros o medios incrustados pueden tener diferentes niveles de accesibilidad

Estamos trabajando activamente para abordar estos problemas.

Comentarios

Agradecemos sus comentarios sobre la accesibilidad de nuestro sitio web. Si encuentra alguna barrera o tiene sugerencias de mejora, por favor contáctenos:

  • Use nuestro formulario de contacto
  • Envíenos un correo electrónico con "Accesibilidad" en el asunto

Nuestro objetivo es responder a los comentarios sobre accesibilidad dentro de 5 días hábiles.

Mejora Continua

Estamos comprometidos con mejoras continuas de accesibilidad. Regularmente:

  • Probamos nuestro sitio web con herramientas de accesibilidad automatizadas
  • Realizamos pruebas manuales con tecnologías asistivas
  • Capacitamos a nuestro equipo en mejores prácticas de accesibilidad
  • Revisamos y actualizamos el contenido para accesibilidad

Fecha

Esta declaración fue actualizada por última vez en enero de 2025.
