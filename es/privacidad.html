Política de Privacidad

Última actualización: Enero 2025

Introducción

We Build Pillars ("nosotros", "nuestro") opera el sitio web webuildpillars.org. Esta página te informa sobre nuestras políticas respecto a la recopilación, uso y divulgación de información personal cuando usas nuestro Servicio.

Información que Recopilamos

Recopilamos varios tipos de información para diversos propósitos para proporcionarte y mejorar nuestro Servicio.

Datos Personales

Al usar nuestro Servicio, podemos pedirte que nos proporciones cierta información personalmente identificable que puede usarse para contactarte o identificarte:

  • Dirección de correo electrónico
  • Nombre y apellido
  • Nombre de la organización
  • Información sobre tu misión y trabajo

Datos de Uso

También podemos recopilar información sobre cómo se accede y usa el Servicio. Estos datos pueden incluir información como la dirección IP de tu computadora, tipo de navegador, versión del navegador, las páginas de nuestro Servicio que visitas, la hora y fecha de tu visita, el tiempo pasado en esas páginas y otros datos de diagnóstico.

Uso de Datos

We Build Pillars usa los datos recopilados para varios propósitos:

  • Para proporcionar y mantener el Servicio
  • Para notificarte sobre cambios en nuestro Servicio
  • Para procesar solicitudes de dominio
  • Para proporcionar soporte al cliente
  • Para detectar, prevenir y abordar problemas técnicos

Seguridad de Datos

La seguridad de tus datos es importante para nosotros, pero recuerda que ningún método de transmisión por Internet o método de almacenamiento electrónico es 100% seguro. Nos esforzamos por usar medios comercialmente aceptables para proteger tus Datos Personales.

Contáctanos

Si tienes alguna pregunta sobre esta Política de Privacidad, contáctanos a través de nuestra página de contacto.
